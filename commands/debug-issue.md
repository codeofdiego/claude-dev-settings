## Debug Production Issue

Systematically investigate and resolve a production issue or bug report.

### Process:
1. **Reproduce the Issue**: Understand steps to reproduce and expected vs actual behavior
2. **Log Analysis**: Search logs, error messages, and stack traces
3. **Code Investigation**: Identify potential root causes in the codebase
4. **Impact Assessment**: Determine severity and affected users/systems
5. **Solution Planning**: Propose fix with minimal risk and proper testing
6. **Hotfix Strategy**: If urgent, plan immediate workaround vs proper fix

### Output:
- Root cause analysis
- Immediate action plan
- Long-term solution recommendation
- Testing strategy before deployment
