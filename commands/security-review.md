## Security Review

Comprehensive security assessment of code, architecture, or specific features.

### Security Areas:
1. **Authentication & Authorization**: Access controls, session management
2. **Input Validation**: SQL injection, XSS, CSRF protection
3. **Data Protection**: Encryption, sensitive data handling, GDPR compliance
4. **Infrastructure Security**: Container security, network policies, secrets management
5. **Dependency Security**: Vulnerability scanning, supply chain security
6. **API Security**: Rate limiting, input validation, OWASP API Top 10

### Assessment Process:
- Threat modeling for the specific component/feature
- Code analysis for common vulnerabilities
- Architecture review for security patterns
- Compliance requirements assessment

### Output:
- Vulnerability assessment with CVSS scores
- Security recommendations prioritized by risk
- Compliance gaps and remediation steps
- Security testing strategy
