## Code Review Assistant

Perform comprehensive code review focusing on quality, security, and best practices.

### Review Areas:
1. **Code Quality**: Readability, maintainability, adherence to conventions
2. **Security**: Vulnerability assessment, input validation, auth/authz
3. **Performance**: Efficiency, scalability concerns, resource usage
4. **Testing**: Coverage, test quality, edge cases
5. **Architecture**: Design patterns, separation of concerns, SOLID principles
6. **Documentation**: Code comments, API docs, README updates

### Output:
- Priority-categorized feedback (Critical/Major/Minor/Suggestion)
- Security concerns with severity ratings
- Performance optimization opportunities
- Missing test cases
- Documentation gaps
